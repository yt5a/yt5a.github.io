・属性の種類

width

スクロールの幅の設定。%やpxで指定。

heigth

スクロールの高さ。%やpxで指定。

hspace

左右の空白。pxで指定。

vspace

上下の空白。pxで指定。

behavior

スクロールの種類を変える。

behavior="scroll"

画面を横切る設定。初期設定。

behavior="alternate"

画面を往復する設定。

behavior="slide"

画面端で停止する設定

direction

スクロール方向の設定。leftかrightで指定。

loop

停止するまでの動作の回数の設定。0を設定した場合は設定なしと同じ動作をする。

scrolldelay

再描画までの時間。指定した数字の値が大きいほど遅く、小さいほど早い。初期値は85。

scrollamount

再描画までの時間。指定した数字の値が小さいほど遅く、大きいほど早い。初期値は85。

scrollamount

再描画までに進む距離の設定。指定した値が大きいほど速く、小さいほど遅い。初期値は6

bgcolor

背景色の設定。

これらの他にIDやクラス、スタイルの属性が使用可能。
